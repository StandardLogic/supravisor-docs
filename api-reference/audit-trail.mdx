# Audit Trail API

The audit trail contains every gate decision as a signed attestation. Records are immutable — each attestation is signed by the gate's private key at the moment of decision and cannot be altered after the fact.

**Base URL:** `https://uniplex.ai/api`

---

## List Attestations

Retrieve attestations with optional filtering.

```
GET /api/audit-trail
```

**Required scope:** `audit:read`

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `agent_id` | string | Filter by agent ID |
| `gate_id` | string | Filter by gate ID |
| `passport_id` | string | Filter by passport ID |
| `decision` | string | `allow` or `deny` |
| `action` | string | Filter by action (e.g., `web:search`) |
| `from` | string | ISO 8601 start timestamp |
| `to` | string | ISO 8601 end timestamp |
| `limit` | integer | Results per page (default: 50, max: 500) |
| `offset` | integer | Pagination offset |
| `sort` | string | `asc` or `desc` by timestamp (default: `desc`) |

**Example Requests:**

```bash
# Last 50 decisions
curl "https://uniplex.ai/api/audit-trail" \
  -H "Authorization: Bearer uni_xxxx" \
  -G --data-urlencode "limit=50"

# All denials for a specific agent
curl "https://uniplex.ai/api/audit-trail" \
  -H "Authorization: Bearer uni_xxxx" \
  -G \
  --data-urlencode "agent_id=my-research-agent" \
  --data-urlencode "decision=deny" \
  --data-urlencode "limit=100"

# All decisions in a time range
curl "https://uniplex.ai/api/audit-trail" \
  -H "Authorization: Bearer uni_xxxx" \
  -G \
  --data-urlencode "from=2026-02-01T00:00:00Z" \
  --data-urlencode "to=2026-02-28T23:59:59Z" \
  --data-urlencode "limit=500"
```

**Response:**

## Create Public Share Link

Create a publicly accessible URL for an attestation. The share URL shows the decision, constraints, and allows signature verification — without exposing API keys or internal gate configuration.

```
POST /api/audit-trail/:attestation_id/share
```

**Required scope:** `audit:share`

**Request Body:**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `expires_in` | string | ❌ | Duration for share link validity (default: `"30d"`) |

**Example Request:**

```bash
curl -X POST https://uniplex.ai/api/audit-trail/att_xyz789/share \
  -H "Authorization: Bearer uni_xxxx" \
  -H "Content-Type: application/json" \
  -d '{"expires_in": "90d"}'
```

**Response:**

```json
{
  "share_url": "https://uniplex.ai/proof/att_xyz789_sharetoken_abcdef",
  "attestation_id": "att_xyz789",
  "expires_at": "2026-05-24T10:30:00Z",
  "created_at": "2026-02-24T10:30:00Z"
}
```

## Export Attestations

Export attestations as JSON or CSV for compliance, auditing, or reporting.

```
GET /api/audit-trail/export
```

**Required scope:** `audit:export`

**Query Parameters:** Same as List Attestations, plus:

| Parameter | Type | Description |
|-----------|------|-------------|
| `format` | string | `json` or `csv` (default: `json`) |
| `include_constraints` | boolean | Include full constraint evaluation (JSON only, default: true) |

**Example Requests:**

```bash
# Export all February 2026 decisions as CSV
curl "https://uniplex.ai/api/audit-trail/export?format=csv&from=2026-02-01&to=2026-02-28" \
  -H "Authorization: Bearer uni_xxxx" \
  -o february-audit.csv

# Export denials for a specific agent as JSON (with full constraints)
curl "https://uniplex.ai/api/audit-trail/export" \
  -H "Authorization: Bearer uni_xxxx" \
  -G \
  --data-urlencode "format=json" \
  --data-urlencode "agent_id=my-research-agent" \
  --data-urlencode "decision=deny" \
  -o agent-denials.json
```

**Response Headers for CSV:**

```
Content-Type: text/csv
Content-Disposition: attachment; filename="audit-trail-2026-02-24.csv"
```

**CSV Columns:**

```
attestation_id,decision,reason,agent_id,agent_name,gate_id,action,resource,timestamp,trust_profile,gate_signature
```

## Get Audit Trail Stats

Aggregate statistics for the audit trail.

```
GET /api/audit-trail/stats
```

**Required scope:** `audit:read`

**Query Parameters:** `agent_id`, `gate_id`, `from`, `to`

curl "https://uniplex.ai/api/audit-trail/stats" \

## Error Responses

| Error Code | HTTP Status | Description |
|-----------|------------|-------------|
| `ATTESTATION_NOT_FOUND` | 404 | No attestation with this ID |
| `SHARE_EXPIRED` | 404 | Share link has expired |
| `EXPORT_TOO_LARGE` | 400 | Export request exceeds 100,000 records — use pagination |
| `INVALID_DATE_RANGE` | 422 | `from` must be before `to` |
