# Constraint Templates

Templates are pre-built guardrail configurations for common agent types. Instead of hand-crafting 17 constraint values from scratch, you pick the profile closest to your use case and you're 90% done. Customize the remaining 10% if needed.

---

## How to Apply a Template

<Steps>
  <Step title="Open Gates in the dashboard">
    Click **Gates** in the left sidebar.
  </Step>
  <Step title="Create a new gate or edit existing">
    Click **+ New Gate**, or click an existing gate → **Edit**.
  </Step>
  <Step title="Choose a template">
    Click **Choose Template** in the Constraints section. Browse by category or search by name.
  </Step>
  <Step title="Apply and customize">
    Click **Apply**. The 17 constraint fields are populated. Adjust any values that don't fit your specific use case.
  </Step>
  <Step title="Save">
    Click **Save Gate**. Your gate is now configured with the template's guardrails.
  </Step>
</Steps>

**Via API:**

```bash
curl -X POST https://uniplex.ai/api/gates \
  -H "Authorization: Bearer uni_xxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Research Gate",
    "profile": "L2",
    "trusted_issuers": ["issuer:mycompany.ai"],
    "template": "web-research-agent"
  }'
```

**Via MCP (tell Claude):**
> "Apply the Web Research Agent template to my Research Gate."

## Customizing After Applying

Templates set the defaults — you own the values. Common customizations:

```bash
# After applying a template, update specific constraints
curl -X PUT https://uniplex.ai/api/gates/gate_def456 \
  -H "Authorization: Bearer uni_xxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "constraints": {
      "max_per_day": "$75.00",
      "domain_allowlist": "myspecificdomain.com,arxiv.org"
    }
  }'
```

Or in the dashboard: **Gates** → your gate → **Edit** → change any constraint value.

## Creating Your Own Template

Once you have a gate configured the way you like, save it as a template:

1. Open the gate → **Edit**
2. Click **Save as Template**
3. Give it a name and description
4. Optional: mark it as **Private** (only your account) or **Team** (shared with your organization)

Custom templates appear in the template browser alongside the built-in 30. They can be applied to new gates just like any other template.
